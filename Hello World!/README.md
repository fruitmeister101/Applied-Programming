## Instructions for Build and Use

Steps to build and/or run the software:

1. Have AutoHotkey-v2 Downloaded and Installed

Instructions for using the software:

1. Run The File
2. Press the UP Arrow Key
3. Press the Down Arrow Key to quit the program

## Development Environment

To recreate the development environment, you need the following software and/or libraries:

* Have AutoHotKey Installed
* Install the AutoHotKey Plus Package for VSCode
